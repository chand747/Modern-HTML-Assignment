HTML vs HTML5

HTML HTML5
HTML does not provide native audio and video support. HTML5 provides native audio and video support.
Longer document type declaration. Shorter document type declaration.
Compatible with almost all browser. Only compatible with newer browser, considering there are many new tags and element which only some browser support.
Less semantic elements, thereby providing less web accessibility features.

It supports more semantic elements, such as <header>, <footer>, <nav>, and <article>, thereby improving accessibility.
